ul vs ol

  • unordered list
    • HTML-tag: ul
    • listet Punkte mit bullet points auf
    • Beispiele:
      1. TO-Do-Liste
      2. Navigation
      3. ...
    • Weiterführende Infos
  • ordered list
    • HTML-tag: ol
    • listet Punkte mit Nummerierung auf
    • Beispiele
      1. Inhaltsverzeichnis
      2. nummerierte Auflistung
      3. ...
    • Weiterführende Infos
